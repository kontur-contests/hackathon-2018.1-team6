1. Начинаем персонажем - "1"

2. Алгоритм выдает цифры в последовательности
	2.1. Базовый уровень - по одной цифре (10 баллов)
	2.2. Продвинутый уровень - по несколько цифр (15 баллов)
	2.3. Для задротов - цифры со скобками (20 баллов)
	2.4. Босс игрового промежутка - тут все серьезно в общем

3. Каждый выход цифр должен проходить проверку на обязательный вариант выигрыша
и считать количество выигрышных ситуаций
	3.1. Если вариантов выигрыша ситуации несколько, то игрок получает 10 баллов
	3.2. Если вариант выигрыша ситуации один, то игрок получает 30 баллов

3. При встрече с врагом необходимо нажать одну из четырех клавиш: * / + -

4. Если ни одна клавиша не нажата - то ПРОИГРЫШ

5. Если клавиша нажата - то запускаем процедуру арифметического действия
	5.1. Отображаем само действие
	5.2. Удаляем ближайшего врага
	5.3. Меняем персонажа на результат арифметической операции
	
6. Если в результате арифм. действия выходим за пределы 0 и 9 - то ПРОИГРЫШ
	6.1. Если наше число (герой) перестает быть целым - то ПРОИГРЫШ

Примеры: 
	1 + <-- 7 = 8		// выводим следующую цифру
	1 + <-- 9 = 10		// проигрыш
	1 / <-- 9 = 0,1		// проигрыш	
	4 + <-- 2*3-1 = 9	// выводим следующую цифру
